Section 4.2 Quotient Rings

¶
Definition 4.2.1.

Let \(R\) be a ring and \(I\) an ideal of \(R\text{.}\) Then elements \(r,s\in R\) are said to be congruent modulo \(I\) if \(b-a\in I\text{.}\) If this is the case, we write \(a + I = b + I\text{.}\)

Question 4.2.2.

Given a ring \(R\text{,}\) ideal \(I\text{,}\) and \(r\in R\text{,}\) when is it the case that \(r + I = 0 + I = I\text{?}\)

Answer

When \(r\in I\text{.}\)

Observe that if \(b-a \in I\text{,}\) then there is some \(x\in I\) such that \(b-a = x\text{,}\) and so \(b = a+x\text{.}\)

Theorem 4.2.3.

Let \(R\) be a ring and \(I\) an ideal of \(R\text{.}\) Then congruence modulo \(I\) is an equivalence relation on \(R\text{.}\)

The set of equivalence classes under this relation is denoted \(R/I\text{.}\)

Proof.

Since \(0\in I\text{,}\) \(a-a \in I\) for all \(a\text{,}\) so \(a + I = a+I\) and the relation is reflexive.

Moreover, if \(b-a \in I\text{,}\) then \(-1(b-a) = a-b \in I\text{,}\) so the relation is symmetric.

If \(b-a \in I\) and \(c-b\in I\text{,}\) then \((b-a) + (c-b) = c-a \in I\text{,}\) so the relation is transitive.

Theorem 4.2.4.

Let \(R\) be a ring and \(I\) an ideal of \(R\text{.}\) If \(a,b,c,d\in R\) such that \(a+I = b+I\) and \(c+I = d+I\text{,}\) then \((a+c) + I = (b+d) + I\text{.}\)

Proof.

Suppose \(a+I = b+I\) and \(c+I = d+I\text{.}\) Then \(b-a =x \in I\) and \(d-c = y \in I\text{.}\) Adding, we have \((b-a) + (d-c) = (b+d) - (a+c) = x+y \in I\text{.}\) Thus, \((b+d) + I = (a+c) + I\text{.}\)

Theorem 4.2.5.

Let \(R\) be a ring and \(I\) an ideal of \(R\text{.}\) If \(a,b,c,d\in R\) such that \(a+I = b+I\) and \(c+I = d+I\text{,}\) then \(ac + I = bd + I\text{.}\)

Proof.

As before, we observe that \(b-a = x\in I\) and \(d-c = y\in I\text{.}\) Write \(b = x+a\) and \(d = c+y\text{.}\) Then \(bd = (x+a)(c+y) = xc + xy + ac + ay = ac + \underbrace{(xc+xy+ay)}_{\in I}\text{,}\) so \(bd - ac \in I\text{,}\) and thus \(bd + I = ac + I\text{.}\)

The previous two theorems together show that addition and multiplication on the set \(R/I\) is well-defined.

Theorem 4.2.6.

Let \(R\) be a commutative ring with identity \(1_R\) and \(I\) an ideal of \(R\text{.}\) The set of equivalence classes modulo \(I\text{,}\) denoted \(R/I\text{,}\) is a commutative ring with identity \(1_R + I\text{.}\)

Checkpoint 4.2.7.

Let \(R = \Z\) and \(I = \ideal{7}\text{.}\) Find at least three distinct integers \(x\) such that \(9 + I = x + I\text{.}\)

Solution

Such \(x\) include \(-5, 2, 16\text{,}\) and anything in \(\setof{2+7m}{m\in\Z}\text{.}\)

Definition 4.2.8.

Let \(R\) be commutative with identity and \(P\subsetneq R\) a nonzero ideal. We say \(P\) is prime if whenever \(a,b\in R\) such that \(ab\in P\text{,}\) we have \(a\in P\) or \(b\in P\text{.}\)

Theorem 4.2.9.

Let \(R\) be a domain and \(p\in R\) be prime. Then \(\ideal{p}\) is a prime ideal.

Checkpoint 4.2.10.

Which of the following ideals are prime?

  1. \(\ideal{9}\) in \(\Z\)

  2. \(\ideal{11}\) in \(\Z\)

  3. \(\ideal{x^2+1}\) in \(\R[x]\)

  4. \(\ideal{x^2-1}\) in \(\R[x]\)

  5. \(\ideal{x^2-5x+6, x^4+2x^3-10x^2+5x-2}\) in \(\R[x]\)

Solution

  1. Not prime. \(3\cdot 3= 9\in \ideal{9}\text{,}\) but \(3\notin\ideal{9}\text{.}\)

  2. Prime. If \(x\in \ideal{11}\text{,}\) then \(11|x\) and 11 is prime.

  3. Prime. Easy explanation is that \(R[x]/\ideal{x^2+1}\) is isomorphic to \(\C\text{.}\) \(\ddot\smile\) For now, though, we know that \(x^2+1\) is irreducible, and \(R[x]\) is a PID, so irreducibles are prime.

  4. Not prime. \(x^2-1 = (x-1)(x+1)\text{,}\) but \(x\pm 1\notin \ideal{x^2-1}\) for degree reasons.

  5. One may use the EA to show that \(\gcd(x^2-5x+6,x^4+2x^3-10x^2+5x-2) = x-2\text{,}\) so \(\ideal{x^2-5x+6, x^4+2x^3-10x^2+5x-2} = \ideal{x-2}\text{.}\) This is prime.

Definition 4.2.11.

Let \(R\) be commutative with identity and let \(M\subsetneq R\) be a nonzero ideal. We say that \(M\) is a maximal ideal if no proper ideal of \(R\) properly contains \(M\text{.}\) That is, if \(J\) is an ideal satisfying \(M\subseteq J\subseteq R\text{,}\) either \(J=M\) or \(J=R\text{.}\)

It is a fact that any ring \(R\) with \(0_R\ne 1_R\) has a maximal ideal. This follows from Zorn's Lemma; a rigorous exploration of Zorn's Lemma lies outside of the scope of these notes, but suffice it to say that Zorn's Lemma is incredibly useful in all areas of algebra for proving existence theorems. For example, a proof that every vector space has a basis relies on Zorn's Lemma.

Rings with only one maximal ideal are said to be local rings, and are actively studied in modern research in commutative algebra (the study of commutative rings and their properties).

Theorem 4.2.12.

Let \(R\) be commutative with identity and \(I\) an ideal of \(R\text{.}\) Then \(I\) is prime if and only if \(R/I\) is an integral domain.

Proof.

Begin by assuming that \(I\) is prime, and suppose \((a+I)(b+I) = 0+I = I\text{.}\) Then \(ab \in I\text{,}\) and since \(I\) is prime, either \(a\in I\) or \(b\in I\text{.}\) Thus either \(a+I = 0+I\) or \(b+I = 0+I\text{.}\)

Now assume that \(R/I\) is a domain. Further, let \(a,b\in R\) be such that \((a+I)(b+I) = ab+I = 0+I\text{.}\) Then \(ab\in I\text{,}\) and since \(R/I\) is a domain, \(a+I = 0+I\) or \(b+I = 0+I\text{,}\) i.e., \(a\in I\) or \(b\in I\text{.}\) Thus, \(I\) is prime.

Lemma 4.2.13.

Let \(R\) be commutative with identity and \(M\) a maximal ideal of \(R\text{.}\) Let \(x\in R\setminus M\text{,}\) and set \(J = \setof{xr+y}{r\in R, \ y\in M}\text{.}\) Then \(M\subsetneq J\text{,}\) and thus there exist \(r'\in R\text{,}\) \(y'\in M\) such that \(1 = xr'+y'\text{.}\)

Theorem 4.2.14.

Let \(R\) be commutative with identity and \(I\) an ideal of \(R\text{.}\) Then \(I\) is maximal if and only if \(R/I\) is a field.

Hint

For the forward direction, apply the previous lemma.

Proof.

If \(R/I\) is a field, assume \(J\) is an ideal of \(R\) that properly contains \(I\text{.}\) Let \(x\in J\setminus I\text{;}\) then \(x+I\) is a nonzero element of \(R/I\text{,}\) and since \(R/I\) is a field, there is some \(y+I\) such that \((xy)+I = 1+I\text{.}\) Since \(x\in J\text{,}\) \(xy\in J\text{.}\) As \(1+I = (x+I)(y+I) = xy+I\text{,}\) we have \(1-xy\in I\subsetneq J\text{,}\) and thus \(1 = (1-xy)+xy \in J\text{,}\) which means \(J = R\text{.}\) Thus, \(I\) is maximal.

Now, suppose that \(I\) is maximal and let \(x\in R\setminus I\text{.}\) Apply the previous lemma to obtain \(1 = xr' + y'\text{,}\) where \(y'\in I\text{.}\) Then

\begin{equation*} 1 + I = xs+y + I = xs + I = (x+I)(y+I)\text{.} \end{equation*}
Theorem 4.2.15.

Every maximal ideal is prime.

Proof.

All fields are integral domains. Thus, if \(I\) is maximal, \(R/I\) is a field, thus a domain, and thus \(I\) is prime.

Theorem 4.2.16.

In a principal ideal domain, every prime ideal is maximal.

Proof.

Let \(R\) be a PID and \(\ideal{p}\) a prime ideal. By previous work, \(p\) is prime. Suppose that \(\ideal{p} \subseteq \ideal{m}\text{.}\) Thus, \(p\in \ideal{m}\text{,}\) so \(m|p\text{.}\) That is, \(p = mk\text{.}\) Since \(p\) is prime and \(R\) is a domain, it is irreducible. Thus, either \(m\) or \(k\) is a unit. If \(m\) is a unit, then \(\ideal{m} = R\text{.}\) If \(k\) is a unit, then \(m = k^{-1} p\text{,}\) and thus \(m\in \ideal{p}\text{,}\) which means that \(\ideal{m} = \ideal{p}\text{.}\)

Thus, \(\ideal{p}\) is maximal.

Find a non-maximal prime ideal.

Solution

In \(\Z[x]\text{,}\) it is straightforward to see that \(\ideal{x}\) is prime. Moreover, \(\ideal{x}\subsetneq \ideal{2,x}\text{,}\) and thus is not maximal.
