Section 4.1 Ideals in general

¶
Theorem 4.1.1.

Let \(R\) be commutative with identity and let \(\set{I_{\alpha}}_{\alpha\in \Gamma}\) be a family of ideals. Then \(I = \bigcap\limits_{\alpha\in \Gamma} I_\alpha\) is an ideal.

Proof.

It is clear that \(0\in I\text{.}\) Moreover, if \(x,y\in I\text{,}\) then \(x,y\in I_\alpha\) for all \(\alpha\text{,}\) so \(x-y\in I_\alpha\) and thus \(x-y\in I\text{.}\) Finally, if \(x\in I\) and \(r\in R\text{,}\) \(rx\in I_\alpha\) for all \(\alpha\text{,}\) and thus \(rx\in I\text{.}\)

We can extend the notion of a principal ideal to ideals generated by any subset.

Definition 4.1.2.

Let \(R\) be a commutative ring with identity, and let \(S\subseteq R\) be a subset. Then

\begin{equation} \langle S \rangle := \bigcap\limits_{\substack{J\supseteq S\\\text{ \(J\) is an ideal } } } J\label{eq_idealgeneratedbyset}\tag{4.1.1} \end{equation}

is called the ideal generated by \(S\).

One way to think of \(\ideal{S}\) is given in the following theorem.

Theorem 4.1.3.

Given a commutative ring \(R\) and a subset \(S\) of \(R\text{,}\) \(\ideal{S}\) is the smallest ideal containing \(S\text{.}\)

Proof.

Let \(I\) be any ideal containing \(S\text{.}\) Thus, \(I\) is one of the ideals on the right-hand side of (4.1.1). Since \(\ideal{S}\) is formed by the intersection of \(I\) with other ideals, \(\ideal{S}\subseteq I\text{.}\)

Theorem 4.1.4.

Given a commutative ring with identity \(R\) and a subset \(S\) of \(R\text{,}\)

\begin{equation*} \langle S \rangle = \setof{r_1 s_1 + r_2 s_2 + \cdots + r_n s_n}{r_i\in R, \ s_j \in S,\ n\ge 0}\text{.} \end{equation*}
Hint

Begin by showing that \(I=\setof{r_1 s_1 + r_2 s_2 + \cdots + r_n s_n}{r_i\in R, \ s_j \in S,\ n\ge 0}\) is an ideal containing \(S\text{.}\)

Proof.

For the ease of notation, let

\begin{equation*} I= \setof{r_1 s_1 + r_2 s_2 + \cdots + r_n s_n}{r_i\in R, \ s_j \in S,\ n\ge 0}\text{.} \end{equation*}

It is clear that \(0\in I\text{,}\) and that if \(r\in R\) and \(x\in I\text{,}\) \(rx\in I\text{.}\) Moreover, the sum of two \(R\)-linear combinations of elements of \(S\) is yet another \(R\)-linear combination of elements of \(S\text{.}\) Thus, \(I\) is an ideal. Further, if \(s\in S\text{,}\) \(1\cdot s \in I\text{,}\) so \(S\subseteq I\text{.}\) Therefore, \(\ideal{S}\subseteq I\text{.}\)

Now assume that \(x\in I\text{.}\) Then \(x\) has the form \(x = r_1 s_1 + \cdots + r_n s_n\text{.}\) Each \(s_i \in S\text{,}\) so each \(r_i s_i \in J\) if \(J\) is any ideal containing \(S\text{.}\) In articular, \(r_i s_i \in \ideal{S}\text{,}\) and thus the sum \(x = \sum r_i s_i \in \ideal{S}\text{.}\)

The ring \(\Z[x]\) is not a PID.

Hint

Consider the ideal \(I = \langle 2, x \rangle\text{.}\)

Proof.

Suppose that \(I = \ideal{a(x)}\text{.}\) Since \(2\in I\text{,}\) \(2 = c(x) a(x)\) for some \(c(x)\in \Z[x]\text{,}\) so \(\deg c(x) = \deg a(x) = 0\text{,}\) and thus \(a(x) = a\in \Z\text{.}\) We also have \(x\in I\text{,}\) so \(x = b(x) a\text{,}\) which implies that \(\deg b(x) = 1\text{,}\) so \(b(x) = b_1 x + b_0\) for some \(b_1,b_0\in \Z\text{.}\) Then \(x = (b_1 x + b_0)a\) implies \(b_1 a = 1\text{.}\) Thus, \(a = 1\) or \(a = -1\text{.}\) In either case, \(a\) is a unit in \(\Z[x]\) so \(1\in I\text{.}\) By Theorem 4.1.4, there are polynomials \(p(x),q(x)\in\Z[x]\) such that \(1 = 2p(x) + xq(x)\text{.}\) If \(p_0\) is the constant term of \(p(x)\text{,}\) it follows that the constant term of \(2p(x) + xq(x)\) is \(2p_0 = 1\text{,}\) which is a contradiction.

Thus, \(I\) is not principal.
