Now, assume that \(a > b\text{.}\) Then the set \(S = \setof{s\in\N}{a-bs \ge 0}\) is nonempty. Further, note that \(S\) has a largest element: every element \(x\in S\) satisfies \(x \le a\text{,}\) since \(a,b\in\N\text{.}\) Let \(q\) be the largest element of \(S\text{,}\) and set \(r = a - bq\text{.}\)
