Definition 2.1.4.

Let \(X\) be a nonempty set. A function \(\star : X \times X \to X\) is called a binary operation. If \(\star\) is a binary operation on \(X\text{,}\) we say that \(X\) is closed under the operation \(\star\). [Given \(a,b\in X\text{,}\) we usually write \(a\star b\) in place of \(\star(a,b)\text{.}\)]
