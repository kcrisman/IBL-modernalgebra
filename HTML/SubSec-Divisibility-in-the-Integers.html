Section 1.2 Divisibility in the integers

¶

In this section, we begin to explore some of the arithmetic and algebraic properties of \(\Z\text{.}\) One of the primary goals of this sections is to formalize definitions that you've been using and have an intuitive understanding of. At first glance, this might seem to add an unnecessary layer of complication. However, it will be abundantly clear as we move forward that a formalized mathematical language and notation are necessary to extend these properties to a more abstract setting. We begin with a twist on a familiar notion.

Definition 1.2.1.

Let \(a,b\in \Z\text{.}\) We say that \(a\) divides \(b\text{,}\) and write \(a\mid b\text{,}\) if there is an integer \(c\) such that \(ac = b\text{.}\) If no such \(c\in \Z\) exists, we write \(a\nmid b\text{.}\)

Note that the symbol \(|\) is a verb; it is therefore correct to say, e.g., \(2|4\text{,}\) as 2 does divide 4. However, it is an abuse of notation to say that \(2\mid 4 = 2\text{.}\) Instead, we likely mean \(4\div 2 = 2\) or \(\frac{4}{2} = 2\) (though we will not deal in fractions just yet).

Investigation 1.2.1.

Determine whether \(a\mid b\) if:

  1. \(a = 3\text{,}\) \(b = -15\)

  2. \(a = 4\text{,}\) \(b = 18\)

  3. \(a = -7\text{,}\) \(b = 0\)

  4. \(a = 0\text{,}\) \(b = 0\)

Theorem 1.2.2.

Let \(a,b\text{,}\) and \(c\) be integers. If \(a\mid b\) and \(a\mid c\text{,}\) then \(a\mid (b+c)\text{.}\)

Proof.

Our hypothesis means that there exist integers \(k_1\) and \(k_2\) such that \(b = a k_1\) and \(c = a k_2\text{.}\) Then \(b+c = a k_1 + a k_2 = a(k_1 + k_2)\text{.}\) Let \(k' = k_1 + k_2\text{,}\) and observe that \(k'\) is an integer; then \(b+c = a k'\text{,}\) so \(a\mid (b+c)\text{.}\)

Theorem 1.2.3.

Let \(a\text{,}\) \(b\text{,}\) and \(c\) be integers. If \(a\mid b\text{,}\) then \(a\mid bc\text{.}\)

Proof.

Our hypotheses means there is an integer \(k\) for which \(b = ak\text{.}\) Then \(bc = (ak)c = a(kc)\text{.}\) We observe that \(k' = kc\) is an integer, so \(bc = ak'\text{,}\) and therefore \(a\mid bc\text{.}\)

Definition 1.2.4.

Let \(a,b\in \Z\) such that \(a\) and \(b\) are not both 0. A greatest common divisor of \(a\) and \(b\text{,}\) denoted \(\gcd(a,b)\text{,}\) is a natural number \(d\) satisfying

  1. \(d\mid a\) and \(d\mid b\)

  2. if \(e\in \N\) and \(e\mid a\) and \(e\mid b\text{,}\) then \(e\mid d\text{.}\)

If \(\gcd(a,b) = 1\text{,}\) we say that \(a\) and \(b\) are relatively prime or coprime.

This formalizes the idea of greatest common factors that is introduced around sixth grade.

This definition may be different than the one you are used to, which likely stated that \(d \ge e\) rather than condition 2 in Definition 1.2.4. It can be proved using the order relations of \(\Z\) that the definition given here is equivalent to that one. However, we will prefer this definition, as it generalizes naturally to other number systems which do not have an order relation like \(\Z\text{.}\)

Activity 1.2.2.

Compute \(\gcd(a,b)\) if:

  1. \(a = 123\text{,}\) \(b = 141\)

  2. \(a = 0\text{,}\) \(b = 169\)

  3. \(a= 85\text{,}\) \(b = 48\)

As we saw above, not all pairs of integers \(a,b\) satisfy \(a\mid b\) or \(b\mid a\text{.}\) However, our experience in elementary mathematics does apply: there is often something left over (a remainder). The following theorem formalizes this idea for \(a,b\in \N\text{.}\)

Theorem 1.2.5. The Division Algorithm.

Let \(a,b\in \N\text{.}\) Then there exist unique integers \(q,r\) such that \(a = bq + r\text{,}\) where \(0 \le r \lt b\text{.}\)

Warning!

This theorem has two parts: existence and uniqueness. Do not try to prove them both at the same time.

Proof.

First, assume that \(a \lt b\text{.}\) Then \(q = 0\) and \(r = a\) are sufficient. If \(a = b\text{,}\) then \(q = 1\) and \(r = 0\) will work.

Now, assume that \(a > b\text{.}\) Then the set \(S = \setof{s\in\N}{a-bs \ge 0}\) is nonempty. Further, note that \(S\) has a largest element: every element \(x\in S\) satisfies \(x \le a\text{,}\) since \(a,b\in\N\text{.}\) Let \(q\) be the largest element of \(S\text{,}\) and set \(r = a - bq\text{.}\)

To finish the proof, it is enough to show that this choice of \(r\) satisfies \(0\le r \le b-1\text{.}\) Observe that \(r\in S\text{,}\) so \(r\ge 0\text{.}\) If \(r \ge b\text{,}\) then \(a - b(q+1) = a-bq -b = r -b \ge 0\text{.}\) Since \(q+1 > q\text{,}\) this is a contradiction to the assumption that \(q\) was the largest element of \(S\text{.}\) Thus, \(r \lt b\text{,}\) i.e., \(r \le b-1\text{.}\)

Let \(q,q',r,r'\) be such that \(a = bq+r\) and \(a=bq'+r'\text{.}\) We see that \(bq+r = bq' + r'\text{,}\) so \(b(q-q') = r'-r\text{,}\) so \(b|r'-r\text{.}\) If \(r'-r = 0\text{,}\) we are done.

Otherwise, \(r'-r\) is a nonzero multiple of \(n\text{.}\) If \(r' \ge r\text{,}\) then \(0\le b(q-q') = r' - r \le b-1 - r\text{,}\) a contradiction. Similarly, if \(r' \le r\text{,}\) write \(b(q'-q) = r-r' \ge 0\text{,}\) and a symmetric argument leads to a contradiction.

Thus, \(r' = r\text{,}\) so \(b(q-q') = 0\text{,}\) and since \(b > 0\text{,}\) \(q - q'= 0\text{,}\) i.e., \(q = q'\text{.}\)

Theorem 1.2.6.

Let \(a,b,c,d\in \Z\text{.}\) If \(a = b+c\) and \(d\) divides any two of \(a,b,c\text{,}\) then \(d\) divides the third.

Proof.

We consider two cases.

Case 1: Assume without loss of generality that \(d|a\) and \(d|b\text{.}\) Then there are integers \(k_1, k_2\) such that \(d k_1 = a\) and \(d k_2 = b\text{.}\) We observe that \(c = a - b = d k_1 - d k_2 = d(k_1 - k_2)\text{,}\) so \(d|c\text{.}\)

Case 2: Assume that \(d| b\) and \(d|c\text{,}\) so there are \(k_1,k_2\in \Z\) such that \(d k_1 = b\) and \(d k_2 = c\text{.}\) Then \(a = b+ c = dk_1 + dk_2 = d(k_1 + k_2)\text{,}\) so \(d|a\text{.}\)

Theorem 1.2.7.

Let \(a,b,n\text{,}\) and \(r\) be integers with \(a\) and \(b\) not both 0. If \(a = n b + r\text{,}\) then \(\gcd(a,b) = \gcd(b,r)\text{.}\)

Proof.

By the previous theorem, if \(d\) is a common divisor of \(a\) and \(b\text{,}\) then \(d\) also divides \(r\text{,}\) and \(d \le \gcd(b,r)\text{.}\) Suppose \(d = \gcd(a,b)\text{,}\) and \(\gcd(b,r)= e > d\) is an integer that divides both \(b\) and \(r\text{,}\) say \(e q_1 = b\) and \(e q_2 = r\text{.}\) Then \(a = n b + r = n (e q_1) + (e q_2)= e (n q_1 + q_2)\text{,}\) so \(e|a\text{,}\) contradicting the assumption that \(d = \gcd(a,b)\text{.}\) Thus \(e = \gcd(b,r) \le d\text{,}\) hence \(e = d\text{.}\)

Activity 1.2.3.

As an illustration of Theorem 1.2.7, note that

\begin{align*} 56 \amp = 4\cdot 12 + 8,\\ 12 \amp = 1\cdot 8 +4,\\ 8 \amp = 2\cdot 4 + 0\text{.} \end{align*}

Use the preceding theorem to show that if \(a = 56\) and \(b = 12\text{,}\) then \(\gcd(56, 12) = \gcd(8, 4) = 4\text{.}\)

Solution

Applying the previous theorem to line 1 tells us that \(\gcd(51,15) = \gcd(15, 6)\text{.}\) The second line tells us that \(\gcd(15,6) = \gcd(6,3)\text{.}\) The third line tells us that \(\gcd(6,3) = \gcd(3,0) = 3\text{.}\) The result follows by transitivity.

Investigation 1.2.4. (Euclidean Algorithm).

Using Theorem 1.2.5 and Theorem 1.2.7, devise a procedure for finding the greatest common divisor of two integers.

Solution

Let \(a,b\in \N\text{.}\) Write \(a = bq_1+r_1\text{.}\) By Theorem 1.2.7, \((a,b) = (b,r_1)\text{.}\)

Then, write \(b = r_1 q_2 + r_2\) using the Division algorithm. Observe \((b,r_1) = (r_1,r_2)\text{.}\) Continue until there is a \(k\) for which \(r_k = 0\text{,}\) so \((r_k,r_{k-1}) = r_{k-1}\text{.}\) Then \(r_{k-1}= (r_k, r_{k-1}) = (r_{k-1},r_{k-2}) = \cdots = (a,b)\text{.}\)

Activity 1.2.5.

Use the Euclidean algorithm to compute \(\gcd(18489,17304)\text{.}\)

Solution

We write

\begin{align*} 18489 \amp = 17304 \cdot 1 +1185\\ 17304 \amp = 1185\cdot 14 +714\\ 1185 \amp = 714 \cdot 1 + 471\\ 714 \amp = 471 \cdot 1+ 243\\ 471 \amp = 243\cdot 1 + 228\\ 243 \amp = 228 \cdot 1+ 15\\ 228 \amp = 15 \cdot 15 + 3\\ 15 \amp = 3\cdot 5 + 0\text{.} \end{align*}

So \(\gcd(18489,17304)=3\text{.}\)

The following identity provides a useful characterization of the greatest common divisor of two integers, not both zero.

Theorem 1.2.8. Bézout's Identity.

For any integers \(a\) and \(b\) not both 0, there are integers \(x\) and \(y\) such that

\begin{equation*} ax + by = \gcd(a,b)\text{.} \end{equation*}
Proof.

If, e.g., \(a = 0\) and \(b\ne 0\text{,}\) then \(\gcd(a,b) = b\) and \(x=0,y=1\) suffices. If \(a = b\text{,}\) then \(\gcd(a,b) = a = b\text{,}\) so \(x = 0, y = 1\) suffices.

So, suppose \(a,b\ne 0\text{,}\) and let \(d = \gcd(a,b)\text{.}\) Without loss of generality assume \(|a| > |b|\text{.}\) The EA provides a sequence of integers \(r_1, r_2, \ldots, r_k, q_1, q_2, \ldots, q_{k+1}\) and equations

\begin{align*} |a| \amp = |b| q_1 + r_1\\ |b| \amp = r_1 q_2 + r_2\\ r_1 \amp = r_2 q_3 + r_3\\ \amp \vdots\\ r_{k-2} \amp = r_{k-1} q_k + r_k\\ r_{k-1} \amp = r_k q_{k+1}\text{,} \end{align*}

where \(0\le r_i \lt |b|\) and \(r_k \ne 0\text{.}\) Then \(r_k = \gcd(a,b)\) and we may back-substitute to find integers \(x\) and \(y\) such that \(ax + by = r_k = \gcd(a,b)\text{.}\)

Theorem 1.2.9.

Let \(a, b\text{,}\) and \(c\) be integers. If \(a|bc\) and \(\gcd(a,b) = 1\text{,}\) then \(a|c\text{.}\)

Proof.

Suppose \(\gcd(a,b) = 1\text{.}\) By Theorem 1.2.8, there are integers \(x\) and \(y\) such that

\begin{equation*} ax+by = 1\text{.} \end{equation*}

Multiply by \(c\) to get

\begin{equation*} acx + bcy = c\text{.} \end{equation*}

Since \(a|bc\) by assumption, there is an integer \(k\) such that \(ak = bc\text{,}\) so we have

\begin{equation*} acx + bcy = acx + aky = a(cx+ky) = c\text{,} \end{equation*}

so \(a|c\text{.}\)
